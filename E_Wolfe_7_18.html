Explain what this script does.

This script:
  1. Assigns a function to the variable "shout".
  2. The function assigns the text "HEY YOU" to a variable named "message"
  3. The script then initializes an alert dialog box to display the text in the variable.
  4. The script then calls the "shout" function which displays the text in the variable "message".
  5. The script then tries to display the text in the variable "message", but since it is outside the scope
    of the function, "message" is no longer defined and nothing happens.
